Place your hand-drawn PNG status icons here. The app now uses separate files for popup buttons and table cells.

Popup/status picker buttons use BLACK icons:
- watched-black.png
- watching-black.png
- dislike-black.png

Table cells on dark backgrounds use WHITE icons:
- watched-white.png
- dislike-white.png

Table cells on light grey/white backgrounds use BLACK icons:
- watching-black.png

Blank / 미선택 uses a question mark in the app, so no PNG is required for blank unless you later want to customize it in code.

Recommended image format:
- PNG
- transparent background
- square canvas, 128x128 or 256x256
- black line art for `*-black.png`
- white line art for `*-white.png`
